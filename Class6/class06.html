Background

All functions in R have at least 3 things:

  • a name (we pick that and use it to call the function)

  • a input arguments (one or more comma separated inputs that go inside the brackets when we call the function),

  • the body (the lines of R code that do the work of the function).

A first function

Here we will create a function to add some numbers. Let’s call it add().

Input arguments can be either “required” or “optional”. The later have fall-back default values that will be used if the user does not specify them.

add <- function(x,y=1000) {
  x + y
}

generate_dna <- function(len=10, single.element = FALSE) {
  dna <- sample(c("A", "C", "G", "T"), size=len, replace = TRUE)
  
  if(single.element) {
    #cat("is it me you are looking for...")
    dna <- paste(dna, collapse = "")
  }
  
  return(dna)
}

generate_protein <- function(len = 9) {
  # Create a vector of the 20 standard amino acids
 aa <- c("A","R","N","D","C","E","Q","G","H","I",
        "L","K","M","F","P","S","T","W","Y","V")

Q5a: At which sequence length do your randomly generated peptides start to look “unique in nature” (i.e. no 100% coverage + 100% identity hit)? [1 pt]

Peptides start to look unique in nature around length 9. This is the first length where there is no longer a 100% identity and 100% coverage match, which means the sequence is no longer an exact match to anything in the database.

Q5b: Speculate why very short random peptides are almost always found in nr, while longer ones typically are not. Your answer should refer both to the size of the sequence space (20𝐿 for a peptide of length 𝐿) and to the size of the known protein universe. [1 pt]

Very short peptides are almost always found because the number of possible sequences is small, so many of them already exist in known proteins. As the length increases, the number of possible sequences grows larger while the known protein database is limited, which makes the longer sequences less likely to match.

Connecting our findings to immunology

Q6: In 3–6 sentences total and using your Q5 data and the reasoning from Q5b, what do you think this minimum length is and why might it be a bad design choice for the immune system to present very short peptides? [3 pt]

MHC class II probably presents peptides that are about 9 amino acids or longer because since that’s when they start to look unique based on the data. Short peptides show up everywhere, so they can’t really tell the difference between your own proteins and foreign ones. As peptides get longer, there are way more possible combinations, so they’re more likely to be unique. If the immune system used really short peptides, it might mistake your own cells as foreign. So using longer peptides helps it see more accurately.
